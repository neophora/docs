# GetHeaderByHeightInJSON

get header by height in json

## Input

```json
{
    "type": "object",
    "properties": {
        "Height": {
            "type": "number",
            "minumum": 0,
            "default": 0,
            "description": "height"
        }
    },
    "required": [],
    "additionalProperties": false
}
```

## Output

```json
{
    "type": "object",
    "description": "extraced header data"
}
```
